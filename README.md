#Projeto Zoológico Web

Projeto desenvolvido durante o 2º semestre da faculdade, com foco em apresentar informações sobre um zoológico.

## Tecnologias
- HTML
- CSS

## Funcionalidades
- Página inicial
- Página de animais
- Contato
- Localização

## Objetivo
Praticar desenvolvimento web e estruturação de páginas.
